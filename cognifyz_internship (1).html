Program Overview

Learn by building.

The Cognifyz Web Developer Internship is a structured, project-based program for aspiring developers. Over 1 to 3 months, interns work alongside experienced engineers on live products and contribute to meaningful codebases.

Whether you're a student or recent graduate, this is your launchpad into the professional tech world — with weekly check-ins, code reviews, and a dedicated mentor.

Who We're Looking For

Desired Skills & Qualifications

You don't need to be an expert — just curious, driven, and ready to learn.

  • Currently pursuing or recently completed a degree in Computer Science, Engineering, or related field
  • Basic understanding of HTML, CSS, and JavaScript fundamentals
  • Familiarity with version control tools like Git and GitHub
  • A proactive attitude and ability to work independently
  • Strong communication skills and willingness to receive feedback
  • A portfolio or personal projects are a plus, but not required
